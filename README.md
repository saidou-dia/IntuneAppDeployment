Intune Application Deployment – Documentation & Procédure

Ce dépôt contient une documentation complète décrivant la méthodologie de déploiement d’applications Windows via Microsoft Intune (Endpoint Manager).
Ce projet a été réalisé dans le cadre d’un stage pour fournir une procédure claire, structurée et directement exploitable par une équipe IT.

Objectifs du projet

    Automatiser et standardiser l’installation des applications Win32, Microsoft Store et Microsoft 365 Apps
    
    Assurer la conformité et la sécurité du parc informatique
    
    Optimiser le temps d’intégration des nouveaux appareils
    
    Fournir une documentation claire et réutilisable
    
    Réduire les erreurs grâce à un guide pas-à-pas illustré
