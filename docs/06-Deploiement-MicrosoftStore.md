# 06 - Déploiement d’applications Microsoft Store

## Objectif
Installer des applications validées disponibles dans le Microsoft Store (ex : Microsoft To Do, OneNote, Remote Desktop).

## Étapes
1. Portail Intune → Applications → Ajouter → Microsoft Store (nouvelle)
2. Rechercher l’application et sélectionner
3. Configurer informations (Nom, Editeur, Description)
4. Affecter groupes utilisateurs/appareils
5. Choisir mode déploiement : Disponible ou Obligatoire
6. Créer et vérifier que l’app apparaît avec statut En attente / En cours / Installée
