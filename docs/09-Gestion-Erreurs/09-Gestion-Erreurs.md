# 09 - Gestion des erreurs courantes

| Erreur           | Cause probable           | Solution                              |
|-----------------|-------------------------|--------------------------------------|
| 0x87D1041C       | App déjà installée       | Vérifier règle de détection           |
| 0x80070002       | Fichier introuvable      | Vérifier chemin de la commande        |
| 0x87D300C9       | Conflit de droits        | Installer en mode administrateur      |
| ArgumentOutOfRangeException | Script de détection incorrect | Corriger règle de détection ou script |
