# 02 - Prérequis

## Poste client
- Windows 10/11 Pro ou Enterprise.
- Appareil Azure AD ou Hybrid Azure AD Join.
- Intune Management Extension installé.
- Accès Internet stable.

## Compte et authentification
- Compte Microsoft 365 avec licence Intune.
- Authentification via Entra ID ou Hybrid Join.
- MFA recommandé pour les admins.

## Rôles et autorisations
- Administrateur Intune
- Administrateur Global

## Outils et environnement
- Portail Intune : [https://endpoint.microsoft.com](https://endpoint.microsoft.com)
- Navigateur moderne : Edge, Chrome, Firefox.
- Applications/packages prêts à déployer (.intunewin, Store, M365 Apps)
