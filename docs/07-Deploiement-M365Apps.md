# 07 - Déploiement de Microsoft 365 Apps (Office)

## Objectif
Installer automatiquement la suite Office (Word, Excel, PowerPoint, Outlook, Teams).

## Étapes
1. Portail Intune → Applications → Ajouter → Microsoft 365 Apps
2. Configuration :
   - Version : M365 Apps for Enterprise
   - Architecture : 64 bits
   - Mises à jour : automatiques
   - Apps incluses : Word, Excel, PowerPoint, Outlook, Teams, OneNote
   - Options : Langue OS, installation silencieuse, mise à jour automatique
3. Affecter groupes utilisateurs/appareils
4. Vérifier déploiement
